HTML Basic Se Seekhne Ka Simple Guide

HTML ka full form hai HyperText Markup Language.

HTML ka use website ki structure banane ke liye hota hai. Example: heading, paragraph, link, image, button.

1. HTML Page Ka Basic Structure

<!DOCTYPE html>
<html>
  <head>
    <title>Mera Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>Yeh mera pehla HTML page hai.</p>
  </body>
</html>
Samjha:
- <html> page ko start karta hai
- <head> page ki information rakhta hai
- <title> browser tab ka title dikhata hai
- <body> page ka actual content hota hai

2. Sabse Common HTML Tags

  • <h1> to <h6> — headings
  • <p> — paragraph
  • <a> — link
  • <img> — image
  • <ul> — unordered list
  • <ol> — ordered list
  • <li> — list item
  • <button> — button
  • <input> — form input box
  • <div> — container

3. Ek Simple Example

<h1>My Name is Ali</h1>
<p>I am learning HTML.</p>
<a href="https://www.google.com">Google</a>

4. List Ka Example

<ul>
  <li>Apple</li>
  <li>Mango</li>
  <li>Banana</li>
</ul>

5. Form Ka Example

<form>
  <label>Name:</label>
  <input type="text">
  <button>Submit</button>
</form>
