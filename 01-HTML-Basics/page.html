HTML Basic Cheat Sheet

1. Basic HTML Structure

<!DOCTYPE html>          <!-- Batata hai ki yeh HTML5 page hai -->
<html>                    <!-- HTML document shuru hota hai -->
  <head>                  <!-- Page ki info (title, CSS, etc.) -->
    <title>Mera Page</title>
  </head>
  <body>                  <!-- Jo dikhega woh yahan likhein -->
    <h1>Hello World!</h1>
  </body>
</html>
Tip: VS Code mein ! type karke Tab dabao — poora structure auto-fill ho jaata hai!

2. Headings (H1 se H6)

<h1>Sabse bada Heading</h1>
<h2>Thoda chhota</h2>
<h3>Aur chhota</h3>
<h4>H4</h4>
<h5>H5</h5>
<h6>Sabse chhota Heading</h6>
Rule: Ek page mein sirf ek <h1> hona chahiye. Baaki sub-headings ke liye h2, h3 use karein.

3. Paragraphs aur Text Formatting

<p>Yeh ek paragraph hai.</p>

<p>Yeh <b>bold</b> text hai.</p>
<p>Yeh <i>italic</i> text hai.</p>
<p>Yeh <u>underline</u> text hai.</p>
<p>Yeh <strong>important bold</strong> text hai.</p>
<p>Yeh <em>emphasized italic</em> text hai.</p>
<p>Yeh <mark>highlighted</mark> text hai.</p>
<p>2<sup>3</sup> = 8  (superscript)</p>
<p>H<sub>2</sub>O  (subscript)</p>

<br>   <!-- Line break (nayi line) -->
<hr>   <!-- Horizontal line (separator) -->

4. Links (Anchor Tag)

<!-- Normal link -->
<a href="https://google.com">Google par jaen</a>

<!-- Naye tab mein khole -->
<a href="https://google.com" target="_blank">Google (naya tab)</a>

<!-- Email link -->
<a href="mailto:aap@example.com">Email karein</a>

<!-- Phone link -->
<a href="tel:+911234567890">Call karein</a>

<!-- Same page ka koi section -->
<a href="#section1">Section 1 par jaen</a>
Attribute Kaam
href Link ka address
target="_blank" Naye tab mein kholo
title Hover par tooltip dikhao

5. Images

<img src="photo.jpg" alt="Meri photo" width="300" height="200">

<!-- Internet se image -->
<img src="https://picsum.photos/300/200" alt="Random photo">
Attribute Kaam
src Image ka path ya URL
alt Image na dikhe to yeh text aata hai
width Chaudai (pixels mein)
height Unchai (pixels mein)
Note: <img> tag ko closing tag ki zaroorat nahi hoti.

6. Lists (Soochi)

<!-- Unordered List (bullets ke saath) -->
<ul>
  <li>Doodh</li>
  <li>Bread</li>
  <li>Ande</li>
</ul>

<!-- Ordered List (numbers ke saath) -->
<ol>
  <li>Paani ubaalen</li>
  <li>Chai patti daalen</li>
  <li>Cheeni daalen</li>
</ol>

<!-- Nested List (list ke andar list) -->
<ul>
  <li>Fruits
    <ul>
      <li>Aam</li>
      <li>Seb</li>
    </ul>
  </li>
</ul>

7. Tables (Taalika)

<table border="1">
  <thead>
    <tr>
      <th>Din</th>
      <th>Subject</th>
      <th>Samay</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Monday</td>
      <td>HTML</td>
      <td>10:00 AM</td>
    </tr>
    <tr>
      <td>Tuesday</td>
      <td>CSS</td>
      <td>11:00 AM</td>
    </tr>
  </tbody>
</table>
Tag Kaam
<table> Table shuru karta hai
<thead> Table ka header section
<tbody> Table ka data section
<tr> Table Row (ek line)
<th> Table Heading (bold)
<td> Table Data (normal cell)

8. Forms

<form action="/submit" method="post">

  <!-- Text Input -->
  <label for="naam">Aapka Naam:</label><br>
  <input type="text" id="naam" name="naam" placeholder="Naam likhein"><br><br>

  <!-- Email Input -->
  <label for="email">Email:</label><br>
  <input type="email" id="email" name="email" placeholder="Email likhein"><br><br>

  <!-- Password -->
  <input type="password" placeholder="Password"><br><br>

  <!-- Radio Buttons -->
  <input type="radio" name="gender" value="male"> Male
  <input type="radio" name="gender" value="female"> Female<br><br>

  <!-- Checkbox -->
  <input type="checkbox" name="agree" value="yes"> I agree<br><br>

  <!-- Dropdown -->
  <select name="city">
    <option value="delhi">Delhi</option>
    <option value="mumbai">Mumbai</option>
  </select><br><br>

  <!-- Textarea -->
  <textarea name="message" rows="4" cols="40" placeholder="Apna message likhein"></textarea><br><br>

  <!-- Submit Button -->
  <button type="submit">Submit</button>
  <button type="reset">Reset</button>

</form>
Input Type Kaam
type="text" Normal text likhne ke liye
type="email" Email validate karta hai
type="password" Text chhupa ke dikhata hai
type="number" Sirf number allow karta hai
type="checkbox" Tick/untick box
type="radio" Ek option chunne ke liye
type="date" Date picker
type="file" File upload

9. Div aur Span

<!-- DIV: Block element — poori line leta hai -->
<div style="background-color: lightblue; padding: 10px;">
  Yeh ek DIV container hai.
</div>

<!-- SPAN: Inline element — sirf apni jagah leta hai -->
<p>Is sentence mein <span style="color: red;">yeh word red</span> hai.</p>
Tag Type Use
<div> Block Sections group karne ke liye
<span> Inline Text ke andar styling ke liye

10. HTML5 Semantic Tags

<header>  ... </header>   <!-- Page ka upar wala hissa -->
<nav>     ... </nav>      <!-- Navigation links -->
<main>    ... </main>     <!-- Main content -->
<section> ... </section>  <!-- Ek section -->
<article> ... </article>  <!-- Blog post ya article -->
<aside>   ... </aside>    <!-- Side content (sidebar) -->
<footer>  ... </footer>   <!-- Page ka neeche wala hissa -->
Yeh tags <div> ki tarah kaam karte hain, lekin inke naam se pata chalta hai ki andar kya hai. Search engines (Google) aur screen readers inhe samajhte hain.

<p style="color: red;">Yeh text red hai.</p>
